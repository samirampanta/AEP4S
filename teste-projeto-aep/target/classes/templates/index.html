ACESSO AO CONHECIMENTO

Software Educacional Gratuito e Multiplataforma (AEP)

Este projeto visa oferecer o acesso democrático e gratuito a conteúdos educacionais variados, como livros e cursos, promovendo a inclusão digital e o aprendizado contínuo. Esta aplicação demonstra a arquitetura em camadas e o gerenciamento de dados persistentes via Java e Banco de Dados H2.
